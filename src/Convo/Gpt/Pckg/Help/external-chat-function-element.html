External GPT Chat Function

Function definition that can be used with Completion API based elements. Unlike the regular chat functions element, this one delegates function execution via Callable parameter.

Function name

Unique function name for identification.

Description

Description about what the function does.

Function parameters

Definitions of all the parameters this function requires.

Defaults

Associative array specifying default values for function parameters.

Required

List of mandatory fields required for this function.

Callable

Variable that hold callable (e.g. function). When function is executed, single parameter (associative array) is passed along, holding all defined parameters.
