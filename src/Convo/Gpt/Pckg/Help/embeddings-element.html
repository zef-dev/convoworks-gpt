GPT Embeddings Element

This element serves as a wrapper for the OpenAI Embeddings API, enabling the retrieval of vector representations for a given input.

Input

The string text for which you want to create an embedding. It's recommended to clean up the raw string beforehand using the provided tokenize_string() function: ${ tokenize_string( your_raw_text)}

Status Variable

The status variable contains the complete response from the embeddings API.

Access the embedding value with: ${status.data[0].embedding}

For more information on available API options, refer to the Embeddings API Reference - OpenAI API.
