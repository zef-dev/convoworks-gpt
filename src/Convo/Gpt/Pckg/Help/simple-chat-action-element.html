Simple Chat Action Element

This element allows you to define action, with it's id and prompt definition, and to define workflow which will be executed when the action is invoked.

Use prompt section to well define action_id, required parameters and expected output. Be sure to also describe in plain language what this action do.

Action Variable Name

This parameter defines name of the variable which will contain action request and be exposed in the element's OK sub-flow.

Result

Expression which evaluates to the action's result data. You will define this valua in the element's OK sub-flow.
