Autonomous Chat Element

This element enables you to create a chat application which will be able to autonomously execute defined actions on your website.
This element uses the completion API.

Conversation

There are three roles in this conversation. Except Bot and User we have a Website agent whcih is able to execute actions.
When the user requests some action, the Bot will respond with plain JSON, stating within which action is requested and passing along all requested params. Chat element will recognize that as the action request, will trigger appropriate action (matched by action_id) and pass the result to the conversation. After that new completion is requested.
Conversation itself (all messages) is handled by the client (service).

Prompt building

Prompt consists of two parts.
The first part is instructions and the chat definition. It is built using the element's Main prompt and prompts defined by the child Prompt elements.
The second part is the current conversation.
While building the Prompts section, you can use IF/ELSE and other control structures, allowing you to dynamically change conversation context.

Actions

Actions are specialized elements, defined with action_id, which are able to execute elements and return the result. Besides Action part, these elements are defining own prompt sections.

Result variable

  • messages array, all messages in the conversation
  • bot_response string, actual bot response
  • last_prompt string, full system message
