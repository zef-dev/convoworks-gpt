GPT Messages Limiter

Limits the size of messages by summarizing the oldest ones. When the conversation reaches the specified maximum count of messages, this component will shrink the conversation by summarizing it down to a specified count.

System Message

The main system prompt instructing GPT-3 on how to summarize the conversation. This prompt helps ensure that the summary contains all the essential details.

Max Messages to Keep

The maximum number of messages that can be present in the conversation before it gets summarized.

Truncate Message Count

The number of messages to retain after the conversation is summarized. This count is less than the maximum messages to ensure the conversation size remains under control.

API Options

Chat completion API options that are used for summarization. These options dictate the behavior of the GPT-3 model, such as which version of the model to use, the desired temperature setting, and the token limit.

Messages

Provides the conversation messages which are being summarized. This is a service component allowing interfaces with the Convo system to retrieve the messages.
