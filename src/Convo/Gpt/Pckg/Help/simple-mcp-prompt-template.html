Simple MCP Prompt Template

Defines a reusable MCP prompt template that you can invoke elsewhere in your pipeline.

Prompt Name

Unique identifier for this template. Can be plain text or an expression wrapped in ${…}.

Description

Short, human‑readable summary of what this prompt does. Displayed in the component preview.

Prompt Arguments

List of argument definitions, wrapped in expression syntax ${…}. Each item must include name, description and optional required flag. Example: ${[{"name":"postId","description":"WP post ID","required":true}]}

Prompt

The actual template text. Reference arguments using ${argumentName}. Supports multi‑line content and any valid expression language constructs.
