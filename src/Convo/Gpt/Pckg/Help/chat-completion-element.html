Chat Completion element

This is OpenAI Chat Completion API wrapper element which allows you to make calls and get out the chat completion API response.

Messages

Messages are array of message objects ({"role":"user|system|assistant", "content":"Some message"}). System message is automatically generated and prepended by the element.
Saving messages through the session is not handled and it is up on the client (service) how it will manage it.

Status variable

Status variable contains the complete chat completion API response.

You can access completion text itself like this: ${status.choices[0]["message"]["content"]}

Check for more on available API options at Chat Completion API Reference - OpenAI API
