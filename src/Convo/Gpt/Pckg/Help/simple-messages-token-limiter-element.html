Simple Messages Token Limiter

Limits the size of the messages array by estimated token count, trimming the oldest messages until the total is under the defined token limit.

Max Tokens to Keep

The maximum estimated token count allowed in the conversation before it gets trimmed. Tokens are estimated for each message's content.

Truncate to Tokens

The estimated token count to retain after trimming the conversation. This value is set lower than the maximum to ensure the conversation size stays manageable.

Result Variable Name

The name of the variable available in the truncated flow. The variable contains two fields: messages, an array containing the current messages, and truncated, an array containing the truncated messages.

Truncated Flow

The flow that is executed when the conversation is truncated.

Messages

Specifies the initial array of conversation messages.
